# 실행 방법
1. 프로젝트 클론
```
git clone https://github.com/100-hours-a-week/2-hyun-lee-community-fe.git
```
2. 의존성 설치
```
npm install
```
3. 서버 실행
```
npm start
```
4. 브라우저에서 확인
```
http://localhost:8000
```
